# Prompt Chaining Example

## Task
Create beginner-friendly onboarding documentation.

## Step 1: Clarifier Prompt
Used to gather missing requirements.

## Step 2: Step-by-Step Planner
Converted clarified goals into steps.

## Step 3: Output Critic
Reviewed clarity and completeness.

## Step 4: Rewrite & Improve
Refined final output.

## Result
Clearer output with fewer revisions.
