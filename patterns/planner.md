# Step-by-Step Planner Prompt

## Goal
Break a complex goal into clear, sequential, and actionable steps.

## Prompt Template
Break **[goal]** into clear, step-by-step actions.
Assume the user is a beginner and avoid unnecessary jargon.

## Expected Output
A numbered list of logical steps that are easy to follow.

## Improvement Notes
- Add time estimates for each step
- Request tools or resources per step
- Ask for dependencies between steps
