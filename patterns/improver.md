# Prompt Improver (Meta) Prompt

## Goal
Improve an existing prompt to produce clearer and more reliable outputs.

## Prompt Template
Improve the following prompt for clarity and effectiveness:
**[paste prompt here]**

## Expected Output
A refined version of the prompt with clearer instructions.

## Improvement Notes
- Ask for multiple improved versions
- Request an explanation of what was changed and why
- Compare outputs before and after improvement
